## Installation

Recommended: pnpm (v9.1.2)
`npm install -g pnpm`

**Install workspace packages**

```sh
pnpm install
```

_without pnpm_: `npm install`

## Start in DEV mode

```sh
pnpm dev
```

Will run a frontend SPA at http://localhost:5173

_without pnpm_: `npm run install`

---

Test the application

```sh
pnpm test:unit
```
